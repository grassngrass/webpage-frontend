# Recruitment Frontend

### Styling Guide

| Name            | Variable        | Hex Code  | Tailwind Class Example          |
| --------------- | --------------- | --------- | ------------------------------- |
| **Background**  | `--background`  | `#0a0a0a` | `bg-background text-foreground` |
| **Foreground**  | `--foreground`  | `#ededed` | `text-foreground`               |
| **Red**         | `--red`         | `#d80607` | `bg-red text-foreground`        |
| **Dark Green**  | `--dark-green`  | `#006044` | `bg-dark-green text-foreground` |
| **Green**       | `--green`       | `#00bc85` | `bg-green text-black`           |
| **Light Green** | `--light-green` | `#79ffd8` | `bg-light-green text-black`     |
| **Black**       | `--black`       | `#111112` | `bg-black text-foreground`      |
